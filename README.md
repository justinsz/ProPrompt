## 🎯 What's Inside?

- **Markdown Crash Course** – The format AI understands best
- **Prompting Fundamentals** – The RICE framework (Role, Instruction, Context, Expected Output)
- **Dos & Don'ts** – 10 best practices + 10 anti-patterns
- **Daily Copilot Usage** – Slash commands, context variables, real-world prompt templates
- **Agent Mode Mastery** – When and how to use autonomous AI agents effectively
- **Copilot Studio Agents** – Build custom agents for Teams, SharePoint & Web
- **Agent Toolchains** – Design multi-agent workflows with orchestration patterns
- **Office → LLM Conversion** – Transform Word, Excel, PowerPoint & PDF into AI-ready formats
- **Project Structure** – Instruction files, custom instructions & context file organization
- **Cheat Sheet** – Copy-paste prompt templates for everyday tasks

## 👥 Who Is This For?

- Developers using GitHub Copilot in VS Code
- Business users building Copilot Studio agents
- Teams setting up AI-powered workflows
- Anyone who wants better results from AI assistants

## 🏁 Quick Start

1. Pick your language: [Deutsch](guide_de.md) or [English](guide_en.md)
2. Start with the **Dos & Don'ts** section for immediate improvements
3. Set up your `.github/copilot-instructions.md` for project-wide AI rules
4. Use the **Cheat Sheet** templates in your daily work
