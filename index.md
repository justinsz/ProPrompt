## 🎯 What's Inside?

Each guide includes:

- **Difficulty levels** from ⭐ Easy to ⭐⭐⭐ Hard
- **Prompt examples** – copy-paste ready for your daily work
- **Agent examples** – Copilot Studio & VS Code Agent Mode
- **Mermaid diagrams** – process flows, toolchains, architectures
- **Cheat sheets** – quick reference templates & context checklists
- **Security guide** – GDPR compliance, data anonymization, AI policies

## 🏁 Quick Start

1. **Lies die Grundlagen / Read the Fundamentals** – RICE-Prinzip, Dos & Don'ts
2. **Wähle deinen Guide / Pick your job guide** – Analysten, Juristen, Entwickler oder Büro
3. **Kopiere die Templates / Copy the templates** – Cheat Sheets für den Alltag
4. **Prüfe den Sicherheits-Guide / Check the Security guide** – Datenschutz & DSGVO
